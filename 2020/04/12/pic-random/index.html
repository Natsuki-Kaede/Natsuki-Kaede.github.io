随机图片API源码&使用

Natsuki Kaede · 2020-4-12 · 加载中··· 次阅读 本文共计：123字

本站提供的接口：https://flymc.cc/2020/08/27/random-cover-api/

GitHub: GitHub - Natsuki-Kaede/Easy-Random-Images-API: A easy random images API.

随机老婆图片接口源码&使用

<?php
$img_array = glob("./图片文件夹/*.{gif,jpg,png}",GLOB_BRACE);
$img = array_rand($img_array); 
header("location:.$img_array[$img]"); 
 ?>

说明：

.php文件应与图片文件夹在同一目录下

{gif,jpg,png} 中可以增加不同的图片格式，如 {gif,jpg,png,bmp,jpeg}

不可部署到静态页面

通过 Domain/*.php调用
